Dropdown Overview

Ornamentum dropdown is an simple dropdown with extended features that a conventional dropdown not supports. as same as the Data Table it was built for focusing on deliver a feature rich, highly configurable,UI framework agnostic, fully responsive and minimal peer dependency support dropdown in top of the Angular Framework.

Peer Dependencies

Ornamentum data table was depend of set of peer dependencies. before library usage, those dependencies need to pre-installed before use. which are,

  • @angular/common
  • @angular/core
  • Lodash-ES
  • RxJS
