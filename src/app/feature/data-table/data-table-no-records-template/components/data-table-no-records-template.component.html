No Records Template

This feature enables users to customize the table body by providing their own template to show when the data is not available, by providing respective mandatory fields for no records template end-user can be notify in a preferred manner in no data scenario.

No records template can be integrate into expandable rows as well. for more details about expandable row configurations please refer Row Expand Template section.

What You Will See

The below demo snippet will demonstrate a sample no data scenario. until a success data response, we have enabled Data Loading Spinner for purpose of visible the process of data retrieval and make the data table more attractive.

Basic Usage

As same as the all other templates, we can bind a specific property #ngDataTableNoRecords into ng-template to indicates we are going to define a template to show when no data is retrieved on data fetch call.

{{demoSnippet.template}}

Or else you can provide your own custom templates with any custom styles as you preferred within the ng-template scope.

{{demoSnippet.stylishTemplate}}
