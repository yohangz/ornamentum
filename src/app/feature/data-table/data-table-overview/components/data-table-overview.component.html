Data Table Overview

Ornamentum Data Grid is an simple data table with extended features that a conventional data table not supports. as same as the Dropdown it was built for focusing on deliver a feature rich, highly configurable,UI framework agnostic, fully responsive and minimal peer dependency support data table in top of the Angular Framework.

Peer Dependencies

Ornamentum data table was depend of set of peer dependencies. before library usage, those dependencies need to pre-installed before use. which are,

  • @angular/common
  • @angular/core
  • Lodash-ES
  • RxJS
