Data Table Row Expand Template

Sometimes you might want to customize the table row expandable section by your their own template. Now this table allows you to provide you own table by simply enabling expandableRows property.

What you will see

Basic Usage

To enable expandable rows, simply set expandableRows to true and by default expandableRows is false.

Data table will not show anything although you have provided true for expandableRows property due to unavailability of a custom template. That means data table doesn't contain any default expandable row template. Therefore, you should provide your own template when you set expandableRows value into true.

Useful properties

There are few more properties available when providing your own custom templates.

Row

row: DataTableRow -> "Data table row properties"

This property can be accessed as shown in the above basic usage sections's example.

Expand On Row Click

expandOnRowClick: boolean -> true or false

Tells data table to expand the table row on user row click event when expandable rows feature is enabled.

Show Row Expand Loading Spinner

showRowExpandLoadingSpinner: boolean -> true or false

Enables loading spinner when data is fetching to the table row expandable section. This can be clear seen when there is scenario to show data by calling a REST API to the expandable row in a slow network.
