Column Sorting

This feature enables column data sorting according to the selected sorting order in specified column fields. Following configurations are available when dealing with data table column sorting.

There are 03 types of sorting orders, which are:

  • Ascending : column data will be sort according to ascending order.
  • Descending : column data will be sort according to descending order.
  • None (Default Sorting Order) : column data will be displayed as it is.

After enabling the sortable users can simply change the sorting order to any above mentioned, by clicking on icon that appears on column top right side corner.

Basic Usage

Sortable can be enabled in any data table column. by default sortable is false to enable column sorting simply set sortable to true.

Useful properties

There are few more properties that affects to the behaviour of column sorting. All the available properties can be applied to each table column by independently excepts Multi Column Sortable property.

Sort Order

sortOrder: string = 'asc' | 'desc' | ''
Default Value: ''

This property allow users to order the data that already fetched or fetching from a data source. If sorting order is not mentioned by the user, then the default sorting mechanism will be applied for those sortable enabled columns.

Sort Field

sortField: string = 'column field name'
Default Value: 'sameColumnFieldName'

Sort Field can be override by providing a column field name, by default sort field value is same to the column name. providing a different column name to sort field, ornamentum will start sorting according to the provided column data.

Multi Column Sortable

multiColumnSortable: boolean = true or false
Default Value: false

Multi column sortable can be only use inside the grid global properties defining scope (ng-data-table). If multi column sortable is enabled, data sorting will be process based on previously preserved column sorting order.
