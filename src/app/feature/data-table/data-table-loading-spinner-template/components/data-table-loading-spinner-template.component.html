Loading Spinner Template

This feature enables users to customize the Data Loading Spinner by providing their own template when data loading is in progress. for enabling Data loading spinner template it is mandatory to enable showLoadingSpinner property in data table global level configuration. for learn more about show loading spinner please refer Data Loading Spinner section.

What You Will See

Basic Usage

As same as the all other templates, we can bind a specific property #ngDataTableLoadingSpinner into ng-template to indicates we are going to define a template by overriding Data Loading Spinner for indicates the on-going data fetch operation.

The following example code block demonstrate the way of data table loading spinner template can be using. In here we uses simple text to override the native loading functionality.

{{demoSnippet.template}}

Or else you can provide your own custom templates with any custom styles as you preferred within the ng-template scope.

{{demoSnippet.stylishTemplate}}

Loading spinner property can be override in module level. for more details about Module level property overriding please refer Module Level Property Override section.
